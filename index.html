Метод Рунге-Кутта 4-го порядка

Численное решение обыкновенных дифференциальных уравнений

Теория метода

Метод Рунге-Кутта 4-го порядка - это один из самых точных и устойчивых численных методов решения задачи Коши. Он позволяет найти приближённое решение дифференциального уравнения вида y' = f(x, y) с начальным условием y(x₀) = y₀.

Формула метода:

yₙ₊₁ = yₙ + h/6·(k₁ + 2k₂ + 2k₃ + k₄)

На каждом шаге вычисляются четыре коэффициента:

k₁ = f(xₙ, yₙ)
k₂ = f(xₙ + h/2, yₙ + k₁·h/2)
k₃ = f(xₙ + h/2, yₙ + k₂·h/2)
k₄ = f(xₙ + h, yₙ + k₃·h)
